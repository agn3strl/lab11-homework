# Домашнее задание к работе 11
## Условие задачи
Сумму всех элементов массива, расположенных до последнего
положительного элемента.
## 1. Алгоритм
1) Начало  
2) Инициализруем массив `A[N]`, счётчик положительных элементов `counter1`   
3) Организуем ввод элементов массива при помощи цикла for. В этом же цикле посчитаем количество положительных элементов массива  
4) Инициализируем переменную `sum` для хранения значения суммы, счётчик `counter2`, который будем сравнивать с `counter1` и продолжать сложение до тех пор, пока он не станет ему равным
5) При помощи цикла for будем прибавлять к `sum` значение текущего элемента массива, если он положительный, то увеличим `counter2`  
6) Выведем на экран значения суммы всех элементов массива, расположенных до последнего положительного элемента  
7) Конец  
## 2. Реализация программы
```C
#define _CRT_SECURE_NO_DEPRECATE
#include <stdio.h>
#include <locale.h>
#include <math.h>
#define N 10

int main()
{
	setlocale(LC_CTYPE, "RUS");

	float A[N];
	int counter1 = 0;
	printf("Введите элементы массива:\n");
	for (int i = 0; i < N; i++) {
		float temp;
		printf("a[%d] = ", i + 1);
		scanf("%f", &temp);
		A[i] = temp;
		if (A[i] > 0) counter1++;
	}

	int counter2 = 0;
	float sum = 0;
	for (int i = 0; counter2 < counter1; i++) {
		sum += A[i];
		if (A[i] > 0) counter2++;
	}

	printf("Сумма всех элементов массива равна %.2f\n", sum);
	return 0;
}
```
